# ML Interview Prep for MANG Companies

Comprehensive ML interview preparation covering topics commonly asked at Meta, Amazon, Netflix, and Google.

## Contents

1. **ML Fundamentals** - Core concepts, bias-variance, regularization, evaluation metrics
2. **Deep Learning** - Neural networks, backprop, CNNs, RNNs, optimization
3. **ML Systems Design** - End-to-end system design, recommendation systems, search ranking
4. **NLP** - Word embeddings, transformers, BERT vs GPT, text classification
5. **Computer Vision** - CNNs, object detection, segmentation, transfer learning
6. **ML Engineering** - Deployment, optimization, monitoring, A/B testing, pipelines
7. **Statistics & Math** - Probability, MLE, gradient descent, hypothesis testing
8. **Reinforcement Learning** - Q-learning, policy gradients, exploration vs exploitation
9. **Coding Problems** - Implement gradient descent, k-means, decision trees, batch norm
10. **Behavioral & Scenarios** - Debugging, data imbalance, cold start, missing data

## How to Use

- Review one topic per day
- Practice coding implementations
- Work through system design scenarios
- Understand the "why" behind each concept
- Focus on trade-offs and real-world applications

## Interview Tips

- Clarify requirements before diving in
- Think out loud during problem solving
- Discuss trade-offs between approaches
- Consider scale, latency, and business impact
- Ask questions about constraints and priorities
- Practice explaining complex concepts simply

Good luck with your interviews!
